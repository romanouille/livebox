aide : sauvegarder



créer une nouvelle sauvegarde
Vous pouvez enregistrer votre configuration actuelle dans un fichier *.bin sur votre ordinateur afin de la restaurer ultérieurement.

Pour enregistrer la configuration actuelle de votre Livebox, cliquez sur le bouton « lancer la sauvegarde » après avoir donné un nom à votre fichier de sauvegarde.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
