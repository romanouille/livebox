aide : NAT/PAT



Certaines applications nécessitent d'ouvrir des ports spécifiques pour fonctionner correctement. Par exemple, si l’on souhaite que des utilisateurs connectés sur Internet puissent accéder à des serveurs de type FTP, messagerie, web etc ...
Afin de diriger certains types de trafic depuis Internet vers une application fonctionnant sur un ordinateur connecté au LAN de la Livebox, il est nécessaire de créer une règle NAT/PAT indiquant le numéro de port et le nom (ou l'adresse IP) de l'ordinateur supportant cette application.
Certaines applications Internet peuvent n'exiger aucun transfert. Quand une demande sur un port bien précis venant d'Internet se présente au niveau du routeur, celui-ci transfère les données vers l'équipement adéquat.
