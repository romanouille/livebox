aide : base téléphone HD



Cette page vous permet de gérer l’association de vos téléphones HD avec votre Livebox et de gérer une liste de contacts qui sera ensuite synchronisée automatiquement avec vos téléphones HD.

liste des téléphones
Vous pouvez associer de 1 à 4 téléphones HD maximum en cliquant sur le bouton « associer un nouveau téléphone ». Une fenêtre « association en cours » s’ouvre, vous pouvez alors lancer la procédure d’enregistrement à partir du combiné. Une fois associé, le téléphone HD apparait dans le tableau avec le nom sous « Téléphone # ». Vous avez alors la possibilité de personnaliser ce combiné en cliquant sur l’icône qui vous redirige vers la page de personnalisation de l’équipement. L’icône poubelle vous permet de désappairer un téléphone HD enregistré.

liste des contacts
Vous pouvez enregistrer des contacts sur cette page d’interface, les contacts seront ensuite immédiatement synchronisés avec tous les téléphones HD enregistrés sur la base téléphone HD. Pour créer un nouveau contact, cliquez sur le bouton « ajouter un contact », une fenêtre s’ouvre, vous pouvez alors saisir le nom et le prénom du contact puis sauvegarder. Le contact apparait ensuite dans le tableau. Pour renseigner les numéros de téléphones, vous devez cliquer à nouveau sur l’icône « voir/modifier », saisir les numéros de téléphones puis sauvegarder. Si vous souhaitez voir tous les contacts sur une seule page, vous devez modifier le nombre de résultats à afficher par page.

modification du code PIN
Par défaut le code PIN est « 0000 ». Vous avez la possibilité de changer ce code PIN, il vous suffit de saisir un nouveau code composé de 4 chiffres. Une fois modifié, ce code PIN vous sera demandé pour toute nouvelle demande d’association de téléphone HD.

remise à zéro de la base du(des) téléphone(s) HD
Vous pouvez réinitialiser la base du(des) téléphone(s) HD. Les paramètres par défaut seront appliqués. Si vous avez modifié le code PIN celui-ci sera réinitialisé à la valeur "0000" et le contenu du journal des appels sera effacé.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
