aide : pare-feu



sélection de votre niveau de protection
Cette fonctionnalité vous permet de protéger plus efficacement votre environnement réseau. Vous avez le choix entre quatre niveaux de protection :
  • faible : votre réseau domestique n’est pas sécurisé, les flux entrants (en direction de votre réseau domestique) et sortants (en direction d'Internet) sont autorisés.
  • moyen : seuls les flux entrants sont bloqués, les flux sortants sont tous autorisés.
  • élevé : tous les flux entrants et sortants sont bloqués. A noter : si vous utilisez le service Unik votre pare-feu ne doit pas être positionné sur le mode élevé.
  • personnalisé : sur la base du pare-feu en mode élevé, vous pouvez configurer vos propres règles.
 protection externe contre les pings
La commande ping permet de tester l'accessibilité d'une machine sur le réseau.
Vous avez la possibilité de répondre ou non à une requête ping envoyée à votre Livebox.
