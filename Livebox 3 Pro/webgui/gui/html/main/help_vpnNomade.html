aide : VPN (pour connexions nomades)



Cette page vous permet de créer de nouveaux groupes de connexion en mode facile ou expert permettant à un utilisateur nomade attaché à ce groupe de se connecter.

paramétrage d'une connexion nomade en mode facile
  • nom du groupe : vous devez définir un nom de groupe (exemple : nomade). Une fois qu’un utilisateur sera attaché à ce groupe, son nom apparaitra dans la section « utilisateurs nomades assignés ».
  • allocation IP : vous pouvez définir la manière dont seront attribuées les adresses IP que les utilisateurs nomades utiliseront une fois connectés à votre site :
    - Livebox DHCP : l’utilisateur nomade obtient une adresse IP automatiquement ;
    - manuel : l’utilisateur nomade configure son logiciel VPN installé sur son ordinateur portable avec une adresse IP choisie dans la plage des adresses IP définies dans la section « configuration DHCP pour utilisateurs VPN nomades » de la page « ma configuration WiFi et Livebox > DHCP et DNS »).
  • accès Internet utilisateur : vous pouvez définir la manière dont l’utilisateur distant accédera à Internet : via sa propre connexion, au travers de la Livebox ou lui interdire l’accès Internet.
  • une clé secrète (clé partagée nomade) qui doit être identique sur la Livebox et le logiciel VPN de l’ordinateur portable de l’utilisateur nomade.
 paramétrage d'une connexion nomade en mode expert
Avec le mode expert, vous pouvez personnaliser les paramètres de sécurité (négociation IKE phases 1 et 2) de votre connexion. Les paramètres affichés par défaut sur la page sont ceux utilisés dans le mode facile. La modification des paramètres de sécurité est réservée aux utilisateurs avancés.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
