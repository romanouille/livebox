aide : restaurer



restaurer une sauvegarde
Cliquez sur le bouton « parcourir » pour rechercher un fichier de configuration enregistré sur votre ordinateur. Cliquez ensuite sur le bouton « restaurer la sauvegarde sélectionnée » pour remplacer la configuration actuelle de votre Livebox par celle du fichier de configuration.

Si le fichier de configuration est conforme, la Livebox va redémarrer. Une fois le redémarrage de la Livebox terminé, vous serez redirigé vers la page d’accueil.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
