aide : VPN (pour connexions site à site)



Cette page vous permet de définir les paramètres de votre connexion VPN site à site.
La configuration en mode facile est à privilégier si vous souhaitez conserver les paramètres de sécurité préconfigurés par Orange.
Le mode expert permet de personnaliser ces paramètres.

paramétrage d'une connexion site à site en mode facile
Pour créer une nouvelle connexion vous devez fournir les informations suivantes :
  • un nom de tunnel (exemple : Paris) ;
  • l’adresse IP (exemple : 193.49.124.107) ou le nom DNS de l’hôte distant, c'est-à-dire l’équipement VPN du site distant (exemple : mondomaine.dyndns.org) ;
  • une clé secrète qui doit être identique sur la Livebox et l’équipement VPN du site distant (exemple : motdepasse) ;
  • la plage des adresses du réseau local et du réseau distant qui auront accès au VPN. Pour ces 2 dernières informations, il faut rentrer une adresse IP ainsi qu’un masque de sous-réseau (exemple : adresse réseau local : 192.168.1.1/24, adresse réseau distant : 192.168.1.2/24).

paramétrage d'une connexion site à site en mode expert
Avec le mode expert, vous pouvez personnaliser les paramètres de sécurité (négociation IKE phases 1 et 2) de votre connexion. Les paramètres affichés par défaut sur la page sont ceux utilisés dans le mode facile. La modification des paramètres de sécurité est réservée aux utilisateurs avancés.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
