aide : test du téléphone relié à la Livebox



En cliquant sur le bouton tester de la page téléphone par Internet, vous avez déclenché le test des téléphones reliés à la Livebox : Livephone (DECT) et/ou téléphone Internet.
S’ils sont correctement connectés à votre Livebox, ils doivent sonner. Dans le cas contraire, merci de bien vérifier les branchements entre les téléphones et la Livebox : le téléphone doit être relié à la Livebox sur la prise Téléphone bleu par un câble de type RJ11. Pour un Livephone (DECT), la base DECT doit être correctement connectée au port USB de la Livebox.
Si après cette vérification, le ou les téléphones ne sonnent toujours pas, merci de contacter le service clients.
Attention, si au moment où le test est lancé, un appel est initié ou reçu, le test est automatiquement abandonné.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
