aide : UPnP



UPnP (Universal Plug and Play) est un protocole réseau automatisant l'ouverture de ports vers les applications compatibles.
La fonction UPnP est activée par défaut, vous pouvez la désactiver si vous ne souhaitez pas que les applications créent automatiquement des redirections de ports.

Si vous souhaitez désactiver/activer cette fonction, n’oubliez pas de cliquer sur le bouton « sauvegarder » pour que le changement soit pris en compte.
Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
