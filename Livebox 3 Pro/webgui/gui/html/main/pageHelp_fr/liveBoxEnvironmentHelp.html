aide : environnement Livebox



Cette page vous permet de visualiser votre environnement Livebox et vous donne accès aux actions disponibles sur la (les) Livebox.

Pour la Livebox principale, les actions disponibles sont :
  • activer ou désactiver WiFi : redirige vers la page « ma configuration WiFi et Livebox > configuration WiFi » qui permet d’activer ou non le WiFi ;
  • redémarrer : affiche la page qui permet de redémarrer la Livebox ou la fonction WiFi ;
  • sauvegarder : affiche la page qui permet de créer une sauvegarde de la configuration de la Livebox sur PC ;
  • restaurer : affiche la page qui permet de restaurer une sauvegarde de la configuration sur la Livebox depuis votre ordinateur ;
  • mettre à jour : affiche la page qui permet de déclencher une mise à jour logicielle de la Livebox ;
  • réinitialiser : affiche la page qui permet de réinitialiser la configuration de la Livebox.
Si la carte mémoire est insérée, 2 actions sont disponibles sur la carte :
  • sécuriser la carte mémoire : si l’opération n’a pas encore été effectuée, redirige vers la page « mon compte > mon mot de passe ». Une fois le mot de passe modifié la carte mémoire sera sécurisée.
  • activer/désactiver le message d’alerte en cas de retrait de votre carte mémoire : redirige vers la page « mon compte > paramètres généraux ». Si vous décidez de désactiver cette alerte, vous ne serez plus notifié de son absence dans la Livebox.
Pour la (les) Livebox secondaire(s), seules les actions de sauvegarde et restauration ne seront pas disponibles.
Dans le cas ou une Livebox secondaire n'est plus active, seule l'opération « retirer de la vue » sera disponible.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
