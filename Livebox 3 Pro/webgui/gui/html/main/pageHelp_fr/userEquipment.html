aide : équipements utilisateurs



Cette page affiche la liste des équipements connectés ou qui ont été connectés à votre Livebox. Vous pouvez ensuite cliquer sur un des ces équipements pour accéder à la page de personnalisation de l'équipement ou pour mettre en place une politique de restriction d'accès à Internet.

Vous pouvez utiliser le filtre d'affichage si vous souhaitez n'afficher qu'un type d'équipement (WiFi, Ethernet, USB, téléphone HD).

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
