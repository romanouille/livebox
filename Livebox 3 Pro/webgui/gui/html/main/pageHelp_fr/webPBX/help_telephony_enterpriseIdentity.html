aide : tableau de bord



mon offre

Cette rubrique présente les caractéristiques de  l’offre commerciale souscrite par l’entreprise :

  • nom de l’offre
  • nombre maximum de postes
  • tranches de  numéros publics SDA (Sélection Directe à l’Arrivée) associée à l’offre. Parmi ces numéros on distingue ceux qui sont disponibles de ceux qui sont déjà affectés  

identité de l'entreprise

L’identité de l’entreprise est définie par un numéro  appelé « numéro entreprise ». C’est un numéro public faisant partie de la tranche de numéros de l’entreprise, mais il peut etre aussi un numéro public spécifique.
Il permet d’acheminer les appels  des postes n’ayant pas de numéro public, et en particulier les appels d’urgence.
Chaque utilisateur ayant un poste avec un numéro public peut néanmoins choisir de présenter le numéro entreprise à la place de son propre numéro, dans le cas des appels externes.
A l’initialisation de l’installation, le numéro entreprise est renseigné automatiquement avec une valeur par défaut. Cette valeur par défaut est le numéro configuré dans les paramètres SIP du faisceau de sortie.

Pour modifier  ce numéro :

  • cliquer sur le lien "modifier le numéro entreprise"
  • l’écran de modification du faisceau de sortie s’affiche
  • sélectionner un  numéro « entreprise »  par défaut dans la liste déroulante correspondante
  • ou bien  sélectionner dans la liste déroulante l’item "entrer un numéro libre" afin de pouvoir saisir un numéro ne faisant pas parti de la tranche des numéros public de l’entreprise 
  • cliquer alors sur le bouton « sauvegarder » pour prendre en compte les modifications effectuées sur cette page.
NB/ le bouton « sauvegarder » se dégrise automatiquement lorsqu’il y a une modification dans la page.

état des téléphones

Cette rubrique présente le nombre de téléphones raccordés à la Livebox ainsi que leur état. Chacun des états est représenté par un icône puis une couleur. On distingue les états suivants avec les icônes correspondants:

  •  postes disponibles
  •  postes occupés
  •  postes  renvoyés vers la messagerie
  • postes renvoyés vers un numéro externe
    •  
  •  postes indisponibles

Pour accéder à la liste des postes

  • cliquer sur le lien « voir liste des téléphones »
  • la liste des téléphones s’affiche

groupements d'appels

Cette rubrique affiche le nombre de groupements de postes disponibles sur le site. Les groupements sont matérialisés par un icône et pour chacun des groupements créés, on a :

  • le nom du groupement
  • le type de groupement
  • les numéros interne et externe du groupement
  • l'état du groupement.

L’état des groupements est représenté comme suit :

  • le  groupement est disponible
  •   le groupement est renvoyé vers la messagerie
  • le groupement est renvoyé vers un numéro

Pour accéder directement à la liste des groupements à partir du tableau de bord

  • cliquer sur le lien « voir liste des groupements »
  • la page contenant la liste des groupements s’affiche
