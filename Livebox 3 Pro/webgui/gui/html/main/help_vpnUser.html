aide : VPN (pour connexions nomades)



Cette page vous permet de créer de nouveaux utilisateurs nomades en leur attribuant un mot de passe personnel afin de pouvoir les authentifier lorsqu’ils voudront se connecter et de leur appliquer un mode de connexion défini au niveau du groupe de connexion auquel ils appartiendront.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
