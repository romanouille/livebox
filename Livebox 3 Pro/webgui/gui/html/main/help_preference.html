aide : paramètres généraux



configuration de la page d'accueil
Cette fonctionnalité vous permet d'afficher ou non la liste des périphériques connectés à votre Livebox sur la page d’accueil.

configuration des alertes
Cette fonctionnalité vous permet :
  • d'activer ou non les messages de confirmation qui s'afficheront à chaque modification effectuée dans les pages de l'interface de configuration de votre Livebox ;
  • d'activer ou non le message d’alerte en cas de retrait de votre carte mémoire.
timeout d’administration
Vous pouvez configurer une durée de session avant expiration entre 10 minutes et 1 heure.
