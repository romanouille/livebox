aide : mon mot de passe



identifiants
Vous pouvez modifier le mot de passe d’administration de votre Livebox. Pour cela vous devez entrer un nouveau mot de passe et le ressaisir dans le champ de confirmation.
Le mot de passe d’administration sert à vous authentifier sur votre Livebox. Il vous est demandé à chaque connexion.
Si votre carte mémoire est insérée dans la Livebox, lors du changement de mot de passe cela vous permet de la sécuriser en même temps.

gestion de mot de passe oublié
En renseignant une réponse à la question secrète, question que vous pouvez modifier, vous avez la possibilité de retrouver votre mot de passe en cas d‘oubli.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
