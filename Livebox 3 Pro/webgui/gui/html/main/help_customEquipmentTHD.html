aide : équipement téléphone HD connecté



personnalisation de l’équipement
Cette section vous permet de personnaliser l'équipement. Vous pouvez lui donner un nom réseau différent du nom par défaut « Téléphone 1 ». Le type d’équipement et son icône ne sont pas modifiables.

connexion de l’équipement
Cette section précise comment l'équipement est connecté à votre Livebox et son statut (connecté ou déconnecté).
Vous avez la possibilité de le supprimer de votre réseau à l’aide du bouton supprimer. Pour le téléphone HD, cette action va le dissocier de la base, il ne sera donc plus enregistré.

Pour que les modifications soient prises en compte, n’oubliez pas de cliquer sur le bouton sauvegarder.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
