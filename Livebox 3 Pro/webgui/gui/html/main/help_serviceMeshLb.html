aide : réseau de Livebox



Cette page vous permet de visualiser l'état d'usage de l'extension WiFi. Elle n’est disponible que lorsqu’une seconde Livebox a été appairée. Le service de réseau de Livebox est alors automatiquement activé.

Vous pouvez désactiver le réseau de Livebox en allant dans la page « ma configuration WiFi et
Livebox > réseau de Livebox » .

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
