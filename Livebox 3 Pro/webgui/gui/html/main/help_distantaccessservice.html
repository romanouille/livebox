aide : accès distant service clients



Pour tout dépannage, l'accès distant service clients permet à un conseiller service clients d'accéder temporairement (15 minutes) à votre Livebox depuis Internet.
Pour lancer cet accès distant service clients, cliquez sur le bouton « lancer ».
Les paramètres suivants : URL / identifiant / mot de passe vont alors apparaître à l'écran. Communiquez-les uniquement au téléconseiller.
Vous pouvez ensuite prolonger cet accès au-delà de 15 minutes, en cliquant sur le bouton « étendre le délai ».
Pour désactiver l'accès distant service clients, cliquez sur le bouton « arrêter ».


Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
