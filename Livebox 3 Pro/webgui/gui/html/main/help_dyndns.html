aide DynDNS



Le service DynDNS vous permet d'associer un nom de domaine et d'hôte fixe à une adresse IP Internet dynamique. Cela peut s'avérer utile si vous hébergez un serveur Web, un serveur FTP ou tout autre type de serveur sur votre réseau local.
  • service : choisissez le fournisseur DynDNS (http://www.dyndns.com/ ou http://www.no-ip.com/ ou http://www.dtdns.com/), auquel vous avez souscrit le nom DynDNS.
  • nom d'hôte complet : saisissez votre nom de domaine.
  • nom d'utilisateur/email : saisissez le nom d’utilisateur correspondant à votre compte DynDNS.
  • mot de passe : saisissez le mot de passe correspondant à votre compte DynDNS.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
