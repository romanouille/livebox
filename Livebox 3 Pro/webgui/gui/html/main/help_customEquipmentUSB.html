aide : équipement USB connecté



personnalisation de l’équipement
Cette section vous permet de personnaliser l'équipement. Vous pouvez lui donner un nom réseau, préciser le type d'équipement, voir son identifiant produit, lui affecter un icône ou encore indiquer son emplacement dans une des pièces de votre entreprise.

connexion de l’équipement
Cette section précise comment l'équipement est connecté à votre Livebox. Le type de connexion et son statut (connecté ou déconnecté).
Vous pouvez aussi rendre l’équipement visible par tout le monde sur le réseau ou au contraire le cacher.

informations sur le périphérique de stockage
Lorsque le périphérique USB est de type stockage, un tableau récapitule les informations suivantes : le nom de la (les) partition(s), le système de fichiers utilisé, la capacité, et l’espace libre. Il vous est possible de lancer une vérification de l’équipement ou bien de le formater.

Si l’équipement est désactivé , vous avez la possibilité de le supprimer de votre réseau à l’aide du bouton supprimer.

Pour que les modifications soient prises en compte, n’oubliez pas de cliquer sur le bouton sauvegarder.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
