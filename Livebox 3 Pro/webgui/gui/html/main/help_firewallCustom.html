aide : personnalisation du pare-feu



Cette fonctionnalité vous permet de personnaliser le pare-feu de votre Livebox. Par défaut, le mode personnalisé reprend toutes les règles pare-feu du mode élevé.

Vous pouvez ajouter des règles de deux façons différentes :
  • soit en appuyant sur le bouton « ajouter » : une nouvelle ligne est ajoutée au tableau des « règles personnalisées », vous devez alors renseigner chaque champ avant de pouvoir sauvegarder cette nouvelle règle ;
  • soit en appuyant sur le bouton « je préfère être guidé » qui vous permet d'être guidé pas à pas pour la construction de cette nouvelle règle.
Il est aussi possible de prioriser ces règles en modifiant leur ordre dans le tableau à l’aide des flèches orange haut et bas . De plus, vous pouvez activer ou désactiver ces règles grâce à la case à cocher « activer ».
L’icône poubelle vous permet de supprimer les règles du tableau. Enfin, le bouton « restaurer » vous permet d’effacer toutes vos modifications et de revenir à la configuration de base (mode élevé).

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
