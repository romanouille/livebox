aide : routage



Pour communiquer, chaque équipement doit connaître le routeur par lequel il faut sortir pour pouvoir atteindre un réseau extérieur, c'est-à-dire avoir en mémoire une table des réseaux et des routeurs. Pour cela, il contient une table de routage locale.
La table de routage d'un routeur comporte les adresses des réseaux de destination, le masque, les adresses des passerelles (routeurs intermédiaires) permettant de les atteindre, l'interface par laquelle le paquet doit sortir de la Livebox.

L'adresse IP saisie dans le champ "réseau de destination" doit correspondre à l'adresse de réseau (la route) pour que la règle de routage fonctionne. Si une adresse IP quelconque appartenant au réseau de destination est saisie, la route est affichée par l'application web mais n'est pas créée et donc ne fonctionnera pas.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
