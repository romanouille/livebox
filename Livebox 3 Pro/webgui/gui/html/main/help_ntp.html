aide : NTP



NTP (Network Time Protocol) est un protocole qui permet de synchroniser, via un réseau informatique, l'horloge locale d'ordinateurs sur une référence d'heure.
Choisissez votre fuseau horaire à partir du menu déroulant.

N’oubliez pas de cliquer sur le bouton « sauvegarder » pour que vos modifications soient prises en compte.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
