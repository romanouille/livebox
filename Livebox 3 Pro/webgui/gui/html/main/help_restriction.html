aide : horaires d’accès à Internet pour l’équipement



Cette page vous permet de limiter l'accès à Internet pour cet équipement en définissant des plages d'accès à l'aide des cases de couleur. Par défaut l’accès est autorisé tous les jours 24h sur 24.

Pour activer une limitation d’accès à Internet sur un jour de la semaine, il suffit de cliquer sur la case correspondante. La case verte devient alors rouge . L'équipement ne peut donc plus accéder à Internet pendant cette plage horaire.

Pour interdire l’accès sur une journée entière, utilisez la case rouge « toujours interdire » à droite du tableau. Inversement pour autoriser l’accès sur une journée, utilisez la case verte
« toujours autoriser ».

Pour que les modifications soient prises en compte, n’oubliez pas de cliquer sur le bouton sauvegarder.

Pour plus d'infos, cliquez sur le lien http://assistance.orange.fr/.
