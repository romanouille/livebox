Téléphone par Internet:

Cette page affiche l’état de votre service de téléphonie par Internet. Le lien vous permet d’accéder directement à votre portail de gestion du service de téléphonie par Internet.
Si le service est activé, la page vous l’indiquera via le message : Le service de téléphonie par Internet est activé. Cela signifie que le service de téléphonie par Internet est opérationnel.
Dans le cas contraire, le message Le service de téléphonie par Internet est désactivé s’affichera. Attendez un peu, si le service ne s’active pas automatiquement, essayez de redémarrer votre livebox. Si le problème persiste, veuillez contacter l’assistance.
Lorsque le service est activé, votre numéro de téléphone par Internet est affiché sur la page. Vous pouvez aussi tester l’état du téléphone relié à votre livebox grâce au bouton Tester.

Pour plus d’infos, cliquez ici (Lien : http://assistance.orange.fr/2626)
