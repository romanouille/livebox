Serveur DHCP:

Vous pouvez utiliser le routeur en tant que serveur DHCP de votre réseau. Un serveur DHCP permet d'attribuer automatiquement une adresse IP à chaque ordinateur de votre réseau.
Adresse IP LAN :
Déterminer l’adresse IP de la livebox et l’adresse qui sera utilisée pour votre réseau domestique. Par défaut, cette adresse est : 192.168.1.1.
Masque de sous-réseau :
Le masque de sous-réseau permet d’identifier un sous réseau. Ici celui de votre réseau domestique.
Début de la plage DHCP:
Saisissez une valeur de début pour la publication d'adresses IP sur le serveur DHCP.
Fin de la plage DHCP :
Déterminez le nombre maximal d'utilisateurs DHCP. Saisissez le nombre maximal d'ordinateurs auxquels le serveur DHCP doit attribuer des adresses IP. Ce nombre ne peut être supérieur à 253.

Adresses IP statiques:

Vous pouvez réserver une adresse IP de votre réseau domestique à un de vos périphériques. Ainsi, ce périphérique aura systématiquement la même adresse IP sur votre réseau. Cette fonctionnalité est utile puisqu’elle vous permet de définir des règles NAT/PAT facilement pour cette machine.
