DynDNS:

Le service DynDNS vous permet d'attribuer un nom de domaine et d'hôte fixe à une adresse IP Internet dynamique. Cela peut s'avérer utile si vous hébergez votre site Web personnel, un serveur FTP ou tout autre type de serveur derrière votre livebox.
Service : Choisissez votre fournisseur DynDNS, auquel vous avez souscrit le nom DynDNS. Hostname : Choisissez votre nom de domaine
Identifiant/e-mail : Saisissez le nom d’utilisateur correspondant a votre compte DynDNS
Mot de passe : Saisissez le mot de passe correspondant a votre compte DynDNS.
