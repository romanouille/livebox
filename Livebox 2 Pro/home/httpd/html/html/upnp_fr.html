UPnP:

Le service UpnP est activé par défaut, vous pouvez le désactiver si vous ne souhaitez pas que les applications créent automatiquement des redirections de ports. Si vous souhaitez désactiver/activer ce service, n’oubliez pas de cliquer sur le bouton enregistrer pour que le changement soit pris en compte.
