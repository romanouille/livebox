Test du Téléphone relié à la livebox:

En cliquant sur le bouton Tester de la page Téléphonie par Internet, vous avez déclenché le test du téléphone relié à la livebox. Si celui-ci est correctement connecté à votre livebox, il doit sonner. Dans le cas contraire, merci de bien vérifier les branchements entre le téléphone et la livebox : le téléphone doit être reliée à la livebox sur le port Téléphone bleu par un câble de type RJ11. Si après cette vérification, le téléphone ne sonne toujours pas, merci de contacter l’assistance. Attention, si au moment où le test est lancé, un appel est initié ou reçu, le test est automatiquement abandonné.

Pour plus d’infos, cliquez ici (Lien : http://assistance.orange.fr/2626)
