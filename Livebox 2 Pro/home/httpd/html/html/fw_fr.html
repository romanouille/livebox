Protection par Pare-feu :

Cette fonctionnalité vous permet de protéger plus efficacement votre environnement réseau. Vous avez le choix entre quatre types de pare-feu :

Faible, votre réseau domestique n’est pas sécurisé, les flux entrants et sortants sont autorisés.
Moyen, seul les flux entrants sont bloqués, les flux sortants sont tous autorisés.
Élevé, tout les flux, entrants et sortants sont bloqués.
Personnalisé, sur la base du Pare-feu en mode Élevé, vous pouvez configurer vos propres règles.
