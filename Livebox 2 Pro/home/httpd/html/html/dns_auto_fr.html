DNS:

Le serveur DNS sert à convertir les noms de domaine en adresses IP et inversement. Vous pouvez configurer manuellement un serveur DNS primaire et secondaire en saisissant leurs adresses IP dans le format X.X.X.X. Si vous choisissez le mode automatique, les serveurs DNS d’Orange seront utilisés par défaut.

Pour plus d’infos, cliquez ici (Lien : http://assistance.orange.fr/2626)
